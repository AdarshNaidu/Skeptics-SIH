Our Approach

The Data Set

The dataset contains the images with normalized difference vegetation index (NDVI) which gives a way to analyse vegetation health based on reflectance values obtained.

The graph

NDVI values for one pixel is analysed over all images giving NDVI values over a period of time. These values are plotted over a period of a year or so. This graph is then smoothened to reduce random factors like instruments errors. This is done over all images to get number of crop cycles.

Output

Color of each pixel in the output image coresponds to the number of harvest cycles.
